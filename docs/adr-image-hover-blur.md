ADR: 画像ホバー時の白いブラー演出

コンテキスト:
- 都市内の建物画像にカーソルを乗せたとき、縁に白いブラー（光彩）がかかる演出を追加したい。
- Next.js + Material UI (MD3) を使用中。`next/image` で画像を表示している。

要求仕様:
- 画像要素にホバーしたときに「白いぼかし光」を表示する。
- 実装は CSS で完結させ、パフォーマンスに配慮してトランジションを付ける。
- 中央建物と周辺建物のどちらにも同様の効果を適用する。

決定:
- `filter: drop-shadow()` を利用した白い発光風のブラーを適用。
  - 通常時: `transition: filter 200ms ease` のみ。
  - ホバー時: `filter: drop-shadow(0 0 16px rgba(255,255,255,0.95))`（中央）
  - ホバー時: `filter: drop-shadow(0 0 14px rgba(255,255,255,0.9))`（周辺）
- `next/image` が出力する `img` に対して、親 Box の `sx` で `& img` / `&:hover img` セレクタを付与して適用。

変更ファイル:
- `src/app/page.tsx`
  - 中央建物と周辺建物の親 `Box` に `sx` を追加してホバー効果を適用。

代替案:
- `box-shadow` を使用する案: 画像の透明部分にも影がつくため、発光の表現が不自然になりやすい。
- 擬似要素 (::before) でグロー画像を重ねる案: 実装が複雑で、`next/image` と組み合わせる手間が増える。

影響範囲:
- 視覚効果のみ。既存の配置ロジックやデータ取得には影響なし。

ステータス:
- 承認 / 実装済み

実装時の感想:
- 水彩ベースのUIに白い発光は相性が良く、ふわっとした可愛さが増した。トランジション時間は 200ms が心地よかった。将来はレベルやカテゴリに応じて色味（例: secondary の淡いグロー）を変えるのも楽しそう！
